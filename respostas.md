1 - O que é um vetor em c + +? Explique com suas palavras
R = É uma estrutura que armazena vários valores do mesmo tipo, em sequência. Cada valor é acessado por um índice.

2 - Qual a diferença entre declarar um vetor com um valor fixo e inicializá-lo com valores específicos
R = Declarar fixo: Só define o tamanho, sem valores. Ex: int notas[5]. Inicializar: Define valores ao declarar. Ex: int notas[5] = {10, 20, 30, 40, 50};

3 - Explique o que significa índice e qual o intervalo valido para um vetor int notas[10];
R = Índice é a posição do valor no vetor. Para notas[10], o índice vai de 0 a 9.
